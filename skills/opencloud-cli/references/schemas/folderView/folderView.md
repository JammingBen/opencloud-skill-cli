# folderView

A collection of properties defining the recommended view for the folder.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `sortBy` | string | No | The method by which the folder should be sorted. |
| `sortOrder` | string | No | If true, indicates that items should be sorted in descending order. Otherwise, items should be sorted ascending. |
| `viewType` | string | No | The type of view that should be used to represent the folder. |
