# fileSystemInfo

File system information on client. Read-write.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `createdDateTime` | string (date-time) | No | The UTC date and time the file was created on a client. |
| `lastAccessedDateTime` | string (date-time) | No | The UTC date and time the file was last accessed. Available for the recent file list only. |
| `lastModifiedDateTime` | string (date-time) | No | The UTC date and time the file was last modified on a client. |
