# thumbnailSet

The ThumbnailSet resource is a keyed collection of thumbnail resources.
It's used to represent a set of thumbnails associated with a DriveItem.


**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | string | No | The ID within the item. Read-only. |
| `large` | [thumbnail](thumbnail.md) | No |  |
| `medium` | [thumbnail](thumbnail.md) | No |  |
| `small` | [thumbnail](thumbnail.md) | No |  |
| `source` | [thumbnail](thumbnail.md) | No |  |
